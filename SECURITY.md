# Security Policy

## Supported Versions

We actively provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to security@rohithealthcare.com. All security vulnerabilities will be promptly addressed.

Please include the following in your report:
- A description of the vulnerability including the potential impact.
- Steps to reproduce the issue (including any necessary code or screenshots).
- Your name and how you would like to be credited.

### Responsible Disclosure Policy

- **Do not** disclose the vulnerability to the public or any third party before we have had a chance to address it.
- **Do not** use the vulnerability to access or modify data that does not belong to you.
- **Do not** perform any "Denial of Service" attacks or other activities that could degrade the service for other users.

We appreciate your help in keeping Rohit Health Care secure for all our patients.
